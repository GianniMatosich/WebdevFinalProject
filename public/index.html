Credits

This application uses local project dependencies only, not CDN-hosted libraries.

The resume builder stores a shared item library in SQLite. Individual resumes save which items are selected for that target role.

  • Bootstrap and Bootstrap Icons for UI styling
  • Node.js, Express, and SQLite for local data storage
  • Electron for the desktop application wrapper
  • Google Gemini API for optional resume suggestions
  • DeepAI Text to Image for the generated logo
